# Логика тайминга News Template

## Общая структура (35 секунд)

```
0 ──────────────────────────────────────────────────────────── 35 сек
│                                                               │
├─ Шапка (4 сек) ─┤
│                 ├─ Ticker 1 (8.67 сек) ─┤
│                 │                        ├─ Ticker 2 (8.67 сек) ─┤
│                 │                        │                        ├─ Ticker 3 (8.67 сек) ─┤
│                 │                        │                        │                        ├─ CTA (5 сек)
│                 │                        │                        │                        │
0                 4                       12.67                   21.34                   30                   35
```

## Детальная разбивка по секундам

### 0-4 секунды: Шапка (Header)
- **Что показывается**: Дата и страна в верхней части экрана
- **Аудио**: Тишина (нет начитки)
- **Видео**: Первое видео начинает играть

### 4-12.67 секунды: Первый Ticker
- **4.00 сек**: Начало показа текста первого ticker'а
- **6.00 сек**: **НАЧАЛО НАЧИТКИ** (через 2 секунды после показа текста)
  - Пользователь успевает перелистнуть видео
  - Начитка голосом Shimmer начинается
- **12.67 сек**: Конец показа текста и начитки
- **Аудио**: Начитка длится 6.67 секунд (8.67 - 2 сек задержки)

### 12.67-21.34 секунды: Второй Ticker
- **12.67 сек**: Начало показа текста второго ticker'а
- **14.67 сек**: **НАЧАЛО НАЧИТКИ** (через 2 секунды после показа текста)
- **21.34 сек**: Конец показа текста и начитки
- **Аудио**: Начитка длится 6.67 секунд

### 21.34-30 секунды: Третий Ticker
- **21.34 сек**: Начало показа текста третьего ticker'а
- **23.34 сек**: **НАЧАЛО НАЧИТКИ** (через 2 секунды после показа текста)
- **30.00 сек**: Конец показа текста и начитки
- **Аудио**: Начитка длится 6.67 секунд

### 30-35 секунды: CTA (Call to Action)
- **Что показывается**: Призыв к действию (изображение)
- **Аудио**: Тишина (нет начитки)
- **Видео**: Все видео остановлены

## Ключевые моменты

1. **Задержка начитки**: 2 секунды после начала показа текста
   - Даёт пользователю время перелистнуть видео
   - Улучшает UX

2. **Длительность начитки**: 6.67 секунд (8.67 - 2)
   - Синхронизирована с показом текста
   - Заканчивается вместе с текстом

3. **Голос**: Shimmer (женский, нейтральный)
   - Профессиональная новостная подача
   - Спокойная и разборчивая речь

4. **Текст на видео**: Без кавычек и квадратных скобок
   - Чистый, читаемый текст
   - Профессиональный вид

## Формулы

- `NEWS_HEADER_DURATION = 4` сек
- `NEWS_TICKER_DURATION = (5 * 6 + 5 - 4 - 5) / 3 = 26 / 3 = 8.67` сек
- `NEWS_CONTENT_DURATION = 4 + 8.67 * 3 = 30` сек
- `TOTAL_DURATION = 30 + 5 = 35` сек

- Время начала текста ticker'а `i`: `4 + i * 8.67`
- Время начала начитки ticker'а `i`: `4 + i * 8.67 + 2`
- Длительность начитки: `8.67 - 2 = 6.67` сек
