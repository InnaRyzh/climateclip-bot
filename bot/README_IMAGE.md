# Где вставить картинку доклада

## Способ 1: Положить файл вручную

1. Положи изображение доклада в папку:
   ```
   bot/assets/images/
   ```

2. Назови файл, например:
   - `report.jpg`
   - `doclad.png`
   - `climate-report.jpg`

3. Изображение будет доступно по URL:
   ```
   http://localhost:3000/assets/images/report.jpg
   ```

4. При рендеринге видео бот автоматически подхватит изображение, если оно есть в папке.

## Способ 2: Загрузить через бота (скоро)

В будущем можно будет отправлять изображение боту, и он сам сохранит его.
